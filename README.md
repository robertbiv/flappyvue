# flappyvue

A Flappy Bird clone built with Vue.js 3 and utilizing Firebase for high score tracking.

## About

This is a browser-based implementation of the popular Flappy Bird game using Vue.js. Click anywhere on the game screen to make the bird jump and navigate through the pipes. Your scores are automatically saved to a Firebase database and displayed on the leaderboard.

## Project setup
```
npm install
```

## How to Play

1. Start the development server with `npm run serve`
2. Open your browser to the local address shown (usually http://localhost:8080)
3. Click anywhere on the game screen to make the bird jump
4. Avoid the pipes and try to get the highest score!
5. Your score will be automatically saved to the leaderboard
